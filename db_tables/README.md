# Задание №1. Спроектировать и создать схему базы данных книжного магазина с использованием SQLAlchemy и PostgreSQL.

---

### Описание таблиц:

> genre - название жанра

> book - название книги, цена, количество на складе, id жанра и автора

> author - автор

> city - город клиента и время доставки

> client - имя клиента, почта, id города

> order - пожелания покупателя и id клиента

> order_book - таблица с количеством заказа покупателей, id книги и заказа

> step - этапы обработки заказа клиента

> order_step - даты начала и окончания этапов обработки заказа с id данными клиента и этапа
