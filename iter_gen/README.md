# Задание 3. Генераторы и итераторы

---

- Что нужно сделать:

**Требуется реализовать итератор последовательности Фибоначчи:**

Нужно реализовать итератор для ленивого вычисления последовательности Фибоначчи до заданного номера без использования генераторов. Класс должен самостоятельно хранить состояние вычислений (текущую позицию и предыдущие значения для вычисления следующего значения). Итератор возвращает числа по запросу и завершает работу при достижении указанного номера.

В учебных целях чудовище, которое получилось, можно потом переписать на генератор.

**Требуется реализовать класс-генератор для постраничного получения данных через API:**

Нужно реализовать генератор в виде класса, то есть метод iter **возвращает генератор**, которые и постранично забирает объекты по API. В репозитории есть "заглушка" для эмуляции запросов к API с постраничным ответом.
