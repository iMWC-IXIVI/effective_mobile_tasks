# Задание 1. Кошелёк

---
> Задание нужно, чтобы поработать с дата-классами и разобраться с магическими методами.

- Что нужно сделать:

**Требуется реализовать простой кошелёк для хранения денег в разных валютах:**

- есть пополнение кошелька деньгами
- есть возможность снятия денег из кошелька
- есть запрет отрицательного баланса по любой валюте (невозможно снять больше, чем имеется)

**Допущения и упрощения**

- конвертация между валютами не учитывается
- при создании кошелька гарантируется неотрицательный начальный баланс по всем валютам
