# Задание 2. Дескрипторы

---

- Что нужно сделать:

**Требуется описать поля модели с использованием дескрипторов:**

- каждое поле должно быть реализовано как отдельный дескриптор
- дескрипторы должны взаимодействовать с данными представленных в виде словаря
- при обращении к полю должно происходить чтение из словаря или запись в словарь

**Допущения и упрощения:**

- случай обновления данных когда отсутствует какой-то ключ можно пренебречь, в тестах есть отдельный тест с меткой skip.
