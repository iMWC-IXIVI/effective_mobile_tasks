# Задание 5. Проектирование применение скидок к заказу

---

- Какой контекст:
> Есть заказ Order, к которому можно применить разные скидки: фиксированную сумму, процент от сумму или скидку за лояльность, для постоянных покупателей. К одному заказу можно применить несколько скидок, причем набор скидок может отличаться для разных заказов и формируется на основе данных, которые хранятся в заказе.

- Что нужно сделать:

Нужно спроектировать функционал для получения и применения скидок к указанному заказу. Система должна включать: класс Order, классы скидок (фиксированная, процентная, за лояльность), а также механизм выбора и применения подходящих скидок.
