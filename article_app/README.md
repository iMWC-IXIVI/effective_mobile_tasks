# Задача 1: Вы разрабатываете приложение для блога. Нужно создать модель Article со следующими полями:

```text
Заголовок (обязательное поле, максимум 200 символов)
Содержание (текстовое поле)
Автор (ссылка на модель User)
Дата публикации (автоматически устанавливается при создании)
Дата последнего обновления (автоматически обновляется при изменении)
Статус публикации (черновик, опубликовано, архив)
Slug (уникальное поле для URL)
```

1. Напишите код модели Article в файле models.py с учетом всех требований.
2. Какие команды нужно выполнить, чтобы создать и применить миграции?
3. Как добавить начальные данные (fixtures) для статусов публикации в базу данных?
