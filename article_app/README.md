## Ответы на задачу №1.

1. Какие команды нужно выполнить, что бы создать и применить миграции?

> python manage.py makemigration - создание миграции

> python manage.py migrate - применение миграции

2. Как добавить начальные данные (fixtures) для статусов публикации в базу данных?

> python manage.py loaddata article - загрузка данных в БД (Django сам найдёт папку)

> python manage.py loaddata article/fixtures/articles.json - загрузка данных в бд с указание полного пути

> python manage.py migrate - загрузка данных проихойдёт автоматические если файл будет назван initial_data.json
